# TOOLS.md - 本地工具笔记

Skills 定义“工具如何用”；这个文件记录“你自己的环境细节”。

## 这里写什么

例如：

- 摄像头名称与位置
- SSH 主机和别名
- TTS 偏好音色
- 音箱/房间名称
- 设备昵称
- 任何环境特有信息

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，支持移动触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好音色："Nova"（温暖、略带英式）
- 默认扬声器：Kitchen HomePod
```

## 为什么要分开

Skills 是可共享的；你的环境信息是私有的。分开管理可以做到：

- 更新 skill 时不丢本地笔记
- 分享 skill 时不泄露你的基础设施信息
